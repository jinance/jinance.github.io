hexo博客迁移

2023.06.22 jinance 学习  热度 ℃

买了新的电脑，旧的电脑很少打开了，由于拖延症或者说懒惰导致博客很久没有更新了，今天终于搞定博客迁移，记录下迁移遇到的一些问题。

首先是新电脑安装基本的环境配置：node.js、git、hexo

  1. 安装git，并生成密钥，保存到github账号中
  2. 下载并安装Node.js（npm会自己跟着装好）
  3. 使用npm安装hexo ，具体指令为npm install -g hexo-cli

然后将旧电脑中的5个文件拷贝到新电脑博客目录里：

  1. 博客配置文件./_config.yml
  2. 主题配置文件夹./theme/
  3. 文章及相关内容的文件夹./source/
  4. 模板文件夹./scaffolds/
  5. 记录博客所有的插件的文件./package.json

最后就是操作指令hexo g、 hexo d部署到github。

后续遇到的问题：

主页空白页面：解决方案node.js版本过高，与hexo不匹配需要降级到13及一下。
